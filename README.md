# IP Finder Application (Java Swing)

## Overview
A desktop-based **IP Finder Application** built using **Java Swing** that allows users to enter a website URL/domain name and retrieve its corresponding IP address instantly.

This project demonstrates the use of Java networking concepts along with GUI development using Swing.

---

## Features
- Accepts URL/domain input from the user  
- Fetches IP address using Java Networking API  
- Displays result in a pop-up dialog box  
- Handles invalid inputs with exception messages  
- Simple and intuitive graphical user interface  

---

## Technologies Used
- Java  
- Java Swing  
- Java Networking (`InetAddress`)  
- NetBeans GUI Builder
